# ML Telegram Bot

## Описание
Telegram-бот для взаимодействия с REST API машинного обучения. Бот позволяет:
- Загружать данные на сервер.
- Предварительно обрабатывать данные.
- Выбирать модель.
- Обучать модель.
- Получать предсказания.

## Технологии
- Python
- python-telegram-bot
- Requests
